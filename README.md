# Phishing Email Detector V4

A real-time cybersecurity tool built with Python that connects to Gmail using the Gmail API, scans inbox emails, and detects suspicious phishing indicators using automated threat analysis.

---

## Features

- Scan live Gmail inbox emails
- Detect suspicious phishing patterns
- Analyze sender and subject lines
- Flag risky emails
- GUI dashboard using Tkinter
- Gmail OAuth authentication
- Automated inbox review
- Resume-level cybersecurity project

---

## Technologies Used

- Python
- Tkinter
- Gmail API
- Google OAuth 2.0
- Regular Expressions
- Machine Learning Ready Architecture
